## 09:08 AKDT — Edge Toolkit Architectural Review + Refactoring

### Architecture Review (self-performed, sub-agent timed out on DeepSeek credits)
- Wrote full review to `memory/edge-toolkit-review.md` (8.4KB)

### Bugs Fixed
1. **jetson-monitor.py**: UnboundLocalError in `read_thermal_zones` — `temp_raw` could be undefined if `f.read()` returned None. Fixed with `rb` mode and explicit init.
2. **edge-rag.py**: Re-indexing appended duplicate chunks forever. Added MD5 hash-based dedup — skips unchanged files, removes old chunks on re-index.
3. **gpu-bench.py**: `get_thermal()` used text-mode `open()` on sysfs nodes. Fixed to `rb` mode with error handling.
4. **edge-gateway.py**: SSE chunk IDs used `int(time.time()*1000)` — duplicates for rapid chunks. Fixed with thread-ID + counter.

### Security Hardening
5. **edge-gateway.py**: API key comparison was timing-vulnerable (`==` on strings). Now uses `hmac.compare_digest()`.
6. **edge-gateway.py**: No request body size limit. Added 1MB cap on POST bodies.
7. **edge-gateway.py**: Default binding was `0.0.0.0`. Changed to `127.0.0.1` (requires `--host` flag for external).

### Architecture Improvements
8. **Created `edge/` shared package** — eliminated code duplication across 4 tools:
   - `edge/config.py` — OLLAMA_URL, paths, limits, chunk sizes

   - `edge/monitoring.py` — thermal/CMA/RAM reading (bug-free)
   - `edge/similarity.py` — cosine similarity + vector ranking
   - `edge/storage.py` — SQLite conversation history + usage tracking
9. **edge-gateway.py**: Upgraded from `HTTPServer` to `ThreadingHTTPServer` (concurrent requests).
10. **edge-gateway.py**: RAG index now cached in memory with mtime-based invalidation.

### New Tools
