## The 2028 Vision

### User Experience (Ring-flash)

- Every chat interaction permanently linked to code changes ("conversation-as-commit")
- Agent suggests architecture based on how she solved similar problems 6 months ago
- Autonomous refactoring learns from manual corrections
- Junior devs get personalized learning paths from accumulated team knowledge
- No prompt engineering — just think out loud while coding

### Emergent Capabilities (DeepSeek-Reasoner)
1. **Predictive Code Metabolism** — maps "metabolic pathways" of code, predicts architectural decay before it happens
2. **Emergent Contributor Empathy** — develops nuanced models of each contributor's rationale and cognitive style, mediates disagreements
3. **Constraint Ghost-Detection** — uncovers invisible, unwritten project constraints that no one documented

### What Emerged Unplanned
- Team knowledge crystallization: predicting which devs will struggle with specific concepts
- The agent acts holistically — absorbing project goals, market shifts, Slack comments
- "Conversation-as-commit" became the primary development artifact

## Backward Chain: 2028 → 2026

### Phase 4: 2028 H1 (The Agent Builds You)
- **Must exist**: Full conversation-as-commit pipeline, autonomous refactoring with human correction loop, cross-repo pattern recognition
- **Critical decision**: Trust the agent to act proactively (the Maya moment)
- **Risk**: Developer alienation ("what do I DO?")

### Phase 3: 2027 H2 (The Repo Wakes Up)
- **Must exist**: Memory seeding (agent works while you sleep), contributor empathy models, constraint ghost detection
- **Critical decision**: Kill the prompt playground — all interaction within development context
- **Risk**: Privacy concerns about tracking all interactions

### Phase 2: 2027 H1 (The Knowledge Crystallizes)
- **Must exist**: Persistent conversation-code linking, historical pattern extraction, personalized learning paths
- **Critical decision**: Store EVERY interaction (keystrokes, file switches, searches) not just chat
- **Risk**: Storage costs, developer resistance to surveillance

### Phase 1: 2026 Q2-Q3 (The Foundation)
- **Must exist**: Basic repo-agent integration, commit-message-aware context, PR annotation from past discussions
- **Critical decision**: Bet everything on "conversation-as-commit" — make chat history a first-class code artifact
- **Risk**: Requires VS Code extension / editor integration
